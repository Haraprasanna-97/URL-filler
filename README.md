# URL filler

## What is this ?

- This is an application is made to fill query parameters in a URL or web link with placeholders
- It also gives its users a chance to test whether the link works as expected after filling the values for the placeholders they have assigned
- The users can see the different parts of the URL if the opt in for that

## How to use

### View URL details 

![Demo of View URL details](https://github.com/Haraprasanna-97/URL-filler/blob/main/GIFs/View%20URL%20details.gif)

- Add the URL in which you want to view the details and hit enter
- Choose the `Show URL details` option from settings
- You will be able to see the URL details

### Get the placeholder URL

![Demo of Get the placeholder URL](https://github.com/Haraprasanna-97/URL-filler/blob/main/GIFs/Get%20the%20placeholder%20URL.gif)

- Add the URL in which you want to add placeholders and hit enter
- In the placeholder names section, provide the names of the placeholders you want to add to the URL. Hit `enter key` to apply the placeholders in the URL
- You can use the copy button to copy the URL with placeholders to the clipboard

### Test the placeholder URL

![Demo of Test the placeholder URL](https://github.com/Haraprasanna-97/URL-filler/blob/main/GIFs/Test%20the%20placeholder%20URL.gif)

- Add the URL in which you want to add placeholders and hit enter
- Provide the placeholder names you want to add in the `Placeholder names` section
- Choose the `Test filled URL` option from settings, you will get the Placeholder values section.
- Provide the values for the placeholders and hit enter. You will get a link with the the values you have added
- Click or tap on that link to test the URL with placeholders
